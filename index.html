누리첵 (NurieCheck)

개인정보처리방침 · 隐私政策

1. 개요

누리첵(이하 "앱")은 눈 건강 자가 체크를 목적으로 하는 비의료 앱입니다. 본 앱은 의료 진단을 제공하지 않으며, 전문 의료기관 방문을 대체하지 않습니다.

본 방침은 앱이 수집·사용·보관하는 정보에 대해 설명합니다.

2. 수집하는 정보

앱은 외부 서버로 개인 정보를 전송하지 않습니다. 아래 데이터는 사용자의 기기에만 저장됩니다.

  • 눈 건강 체크 결과 — 일일 자가 체크 점수 및 개별 테스트 결과 (최대 90일)
  • 앱 설정 — 언어 선택, 알림 활성화 여부, 알림 시간
  • 사용 기록 — 체크 날짜 및 연속 사용 일수(스트릭)

3. 권한 사용 목적

  • POST_NOTIFICATIONS — 매일 눈 건강 체크 알림 전송
  • SCHEDULE_EXACT_ALARM / RECEIVE_BOOT_COMPLETED — 기기 재시작 후에도 지정된 시간에 정확한 알림 유지
  • VIBRATE — 알림 진동

4. 제3자 공유

본 앱은 사용자 데이터를 제3자와 공유하거나 판매하지 않습니다. 모든 데이터는 기기 로컬 저장소(SharedPreferences)에만 보관됩니다.

5. 데이터 보관 및 삭제

체크 기록은 최대 90일분만 보관됩니다. 설정 화면의 데이터 초기화 기능으로 모든 데이터를 즉시 삭제할 수 있습니다. 앱을 삭제하면 모든 로컬 데이터가 함께 삭제됩니다.

6. 아동 개인정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 의료 면책 조항

본 앱은 의료기기가 아니며, 제공되는 모든 정보는 자가 체크 목적에 한합니다. 이상 증상이 있을 경우 반드시 전문 안과 의료기관을 방문하시기 바랍니다.

1. 概述

누리첵（以下简称"本应用"）是一款用于眼部健康自测的非医疗应用。本应用不提供医疗诊断，不能替代专业医疗机构的诊断与治疗。

本政策说明本应用收集、使用和存储的信息。

2. 收集的信息

本应用不会向外部服务器传输任何个人信息。以下数据仅存储在用户设备本地。

  • 眼部健康检测结果 — 每日自测评分及各项测试结果（最多保留90天）
  • 应用设置 — 语言选择、通知开关、通知时间
  • 使用记录 — 检测日期及连续使用天数

3. 权限使用说明

  • POST_NOTIFICATIONS — 发送每日眼部健康检测提醒
  • SCHEDULE_EXACT_ALARM / RECEIVE_BOOT_COMPLETED — 设备重启后仍能在指定时间准时发送通知
  • VIBRATE — 通知震动

4. 第三方共享

本应用不与任何第三方共享或出售用户数据。所有数据仅保存在设备本地存储（SharedPreferences）中。

5. 数据保留与删除

检测记录最多保留90天。您可以通过设置页面的数据重置功能立即删除所有数据。卸载应用时，所有本地数据将被一并删除。

6. 儿童隐私

本应用不面向14岁以下儿童，不会故意收集儿童的个人信息。

7. 医疗免责声明

本应用不是医疗器械，所提供的所有信息仅供自我检测参考。如有异常症状，请务必前往专业眼科医疗机构就诊。
